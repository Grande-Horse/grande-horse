# 🐎 그런데 말입니다
실제 한국마사회의 경주마 데이터에 기반한 2D 도트 그래픽 경마 게임 👾

## 🚀 주요 기능
### ➊ 경마 게임 및 실시간 채팅 💬
- 경주마 능력치와 클릭 이벤트에 기반한 멀티 레이싱 게임 
- 경마방을 생성하고 다수의 유저와 실시간 소통

### ➋ 말 카드 획득 및 합성 기능 🐴
- 실제 마사회의 경주마 데이터를 바탕으로 말의 등급과 능력치 측정
- 말 카드를 합성하여 일정 확률로 더 높은 등급의 말 카드 획득
